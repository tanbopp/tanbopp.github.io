Overview

Konsep kemasan ini bertujuan menghadirkan pengalaman unboxing yang bersih dan modern, menonjolkan fungsionalitas sekaligus estetika produk. Visual dirancang agar mudah diterapkan pada berbagai format kemasan (box, label, insert).

Objective

  • Meningkatkan daya tarik rak dan klik di marketplace
  • Memastikan informasi produk tersampaikan jelas
  • Mengoptimalkan desain untuk produksi massal

Concept

Menggabungkan elemen tipografi tegas dengan ruang putih yang luas untuk menciptakan kesan premium. Ikonografi sederhana dipakai untuk mempercepat pemahaman fitur produk.

Materials & Specs

Karton kraft 300gsm (primary box), matte lamination, cetak offset CMYK + spot varnish untuk aksen logo. Ukuran: S / M / L (disesuaikan dengan SKU).

Color Palette

Typography

Inter (Display) untuk judul — berani dan bersih. Inter (Regular) untuk deskripsi dan informasi teknis. Ukuran heading H1 28–32px, body 14px.

Production Notes

  • Pastikan bleed 3mm, safe area 5mm
  • Gunakan warna Pantone jika diperlukan untuk akurasi warna
  • Sediakan file dieline terpisah untuk setiap ukuran

Deliverables

File desain: AI, PDF produksi, PNG presentasi, dan guideline singkat untuk vendor.
